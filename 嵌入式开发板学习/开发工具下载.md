###  arduino 修改Arduino IDE 的 **全局数据目录**

`Arduino15` 数据目录的选项。`Arduino15` 是 Arduino IDE 的 **全局数据目录**，默认在：

```
C:\Users\<用户名>\AppData\Local\Arduino15
```

如果用户名是中文，很多插件或板子包（比如 ESP32）就可能出问题。解决方法有两个可行方案：

------

### **方法1：通过环境变量修改 Arduino15 目录**

1. 新建一个英文路径，比如：

   ```
   D:\ArduinoData
   ```

2. 设置环境变量：

   - 搜索 **“环境变量” → 编辑系统环境变量 → 环境变量”**

   - 在 **用户变量** 中新建：

     > **注意:**  是用户变量不是系统变量

     ```
     变量名：ARDUINO_HOME
     变量值：D:\ArduinoData
     ```

   - 或者新建 `ARDUINO15` 环境变量（某些版本支持）

3. 重启 Arduino IDE，IDE 会自动在新路径下生成 `Arduino15` 数据。

> 注意：不是所有 Arduino IDE 版本都支持用环境变量修改全局数据目录，但 Arduino IDE 2.x 版本一般可以识别 `ARDUINO_HOME` 或 `ARDUINO15`。
