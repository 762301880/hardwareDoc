#  开发板购买

##  esp32-s3

> 这块板子是 **嘉顺 ESP32-S3-N16R8 开发板**，属于一款性能很强、通用性高的版本。
> 它的核心模块是 **ESP32-S3-WROOM-1-N16R8**（16MB Flash + 8MB PSRAM），适合做 **AI + IoT** 项目。

<img src="https://gitee.com/yaolliuyang/blogImages/raw/master/blogImages/image-20251101103431746.png" alt="image-20251101103431746" style="zoom: 33%;" />

###  板子主要特性

| 项目     | 说明                      |
| -------- | ------------------------- |
| 芯片     | ESP32-S3，双核 Xtensa LX7 |
| Flash    | 16 MB                     |
| PSRAM    | 8 MB                      |
| 无线     | Wi-Fi 4 + 蓝牙 5.0        |
| 电源接口 | USB Type-C × 2            |
| GPIO     | 36 个可编程引脚           |
| 工作电压 | 3.3 ~ 5 V                 |

> 板子上两个 Type-C 口：
>
> - **左边（标“USB”或“UART”）**：用于下载程序和串口调试。
> - **右边（标“USB-OTG”）**：用于让 ESP32-S3 自己作为 USB 设备（比如模拟键盘、摄像头等）。
>    新手先用左边那个口即可。
