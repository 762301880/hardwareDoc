#  开发板购买

##  esp32-s3

> 这块板子是 **嘉顺 ESP32-S3-N16R8 开发板**，属于一款性能很强、通用性高的版本。
> 它的核心模块是 **ESP32-S3-WROOM-1-N16R8**（16MB Flash + 8MB PSRAM），适合做 **AI + IoT** 项目。

###  板子主要特性

| 项目     | 说明                      |
| -------- | ------------------------- |
| 芯片     | ESP32-S3，双核 Xtensa LX7 |
| Flash    | 16 MB                     |
| PSRAM    | 8 MB                      |
| 无线     | Wi-Fi 4 + 蓝牙 5.0        |
| 电源接口 | USB Type-C × 2            |
| GPIO     | 36 个可编程引脚           |
| 工作电压 | 3.3 ~ 5 V                 |

**两个usbc口介绍**

看背面右边 这个为**com**   左边的为**usb**

根据你观察到的板载标识（左侧印 “USB”，右侧印 “COM”），嘉顺 ESP32-S3-N16R8 两个接口的功能可以明确区分如下：

1. **左侧标注 “USB” 的接口**这是 **原生 USB 接口**，直接对应 ESP32-S3 芯片的 USB 硬件模块（无额外转换芯片），主要功能是：

   - 支持 **USB OTG 模式**：可作为主机（连接 U 盘、键盘等外设）或从机（模拟 U 盘、HID 设备等）；
   - 部分场景下可作为 “USB 设备模式” 与电脑通信（如虚拟串口，但需软件配置）；
   - 通常也可为主板供电（5V 输入），但核心功能是扩展 USB 外设交互。

2. **右侧标注 “COM” 的接口**“COM” 即 “串口”，该接口是 **调试与下载专用接口**，内置了 USB-to-UART 转换芯片（如 CH340 或 CP2102），功能包括：

   - 连接电脑进行程序烧录（下载固件到 ESP32-S3）；

   - 通过串口（UART）输出调试日志（如 `Serial.print()` 信息）；

   - 为主板供电（开发阶段最常用的供电方式）。

     

     这是开发时最核心的接口，直接关联芯片的调试串口，必须通过它完成初始程序下载和调试。

**总结**：

- 用 “COM” 接口连接电脑，用于烧录程序、看调试日志（开发必备）；

- 用 “USB” 接口实现 OTG 功能，连接外部 USB 设备或模拟 USB 外设（功能扩展）。

  按标识使用即可，避免混淆。
